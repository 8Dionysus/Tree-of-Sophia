## Current wording under pressure

Current canonical node:

- `tree/principle/friedrich-nietzsche/thus-spoke-zarathustra/prologue-1/blessing-is-reciprocal/node.json`

Current distilled claim:

- blessing appears as reciprocal rather than unilateral because overflow and
  reception answer one another within the same bounded movement

The pressure point is not that the current wording is false.
The pressure point is that `reciprocal` can be misread as:

- reactive response
- symmetrical exchange
- moral payback
- two-sided equivalence

That is narrower and more transactional than the route may later support.
