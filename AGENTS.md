## Project identity

Tree of Sophia (ToS) is a living knowledge architecture for philosophy and world thought.

This repository is currently a high-level canonical statement of ToS, not yet the complete implementation home.

## Ecosystem map

- `Tree-of-Sophia` = knowledge architecture
- `Agents-of-Abyss` = agent ecosystem that supports ToS operations and growth
- `abyss-stack` = local/hybrid AI infrastructure backbone
- `aoa-techniques` = reusable techniques relevant to knowledge and agent workflows
