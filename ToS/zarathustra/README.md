# Zarathustra Route

`ToS/zarathustra/` is the golden route branch for the current Tree of Sophia
axis.

It keeps the first bounded *Thus Spoke Zarathustra* entry legible without
turning the route capsule into source witness, canon, generated export, or
mechanics.

## Map

| Path | Use For |
| --- | --- |
| `prologue-1/` | worked route capsule for the current bounded prologue entry |
| `public-entry/` | tiny-entry route surface for public and low-context entry |

## Route

`source-witnesses/ -> zarathustra/ -> canon/ -> public-compatibility/ -> derived-exports/`

The source text stays in `ToS/source-witnesses/`.
The authored source node stays in `ToS/canon/`.
The compatibility example stays in `ToS/public-compatibility/`.
The generated downstream read model stays in `ToS/derived-exports/`.
