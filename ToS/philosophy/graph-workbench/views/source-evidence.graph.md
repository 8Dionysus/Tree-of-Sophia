# Source Evidence Graph View

## Lens

Shows source witnesses, evidence status, preservation posture, controversy,
and proof limits.

## Future Inputs

- source-witness nodes
- artifact nodes
- preservation nodes
- controversy nodes
- `attested-in`, `evidence-for`, `preserved-by`, and `contested-by` relations

## Boundary

This view makes evidence visible. It does not make research packets into source
witnesses.
