# Imperial Multilingualism Graph View

## Lens

Shows parallel language versions, administrative language regimes,
translation layers, versioned inscriptions, and empire-scale writing systems.

## Future Inputs

- language and script nodes
- version nodes
- inscription, decree, archive, and corpus nodes
- `versioned-as`, `translated-through`, `written-in`, `uses-script`, and
  `institutionalized-in` relations

## Boundary

This view shows multilingual order. It does not make every version equal in
authority, audience, or source posture.
