# Script Decipherment Graph View

## Lens

Shows scripts, sign systems, artifacts, partial readings, decipherment routes,
and uncertainty posture.

## Future Inputs

- script nodes
- artifact and inscription nodes
- decipherment nodes
- controversy nodes
- `deciphered-by`, `partly-deciphered-as`, `attested-in`, and `contested-by`
  relations

## Boundary

This view protects uncertainty. It should not turn a frontier script into a
stable philosophical corpus before source evidence can bear that claim.
