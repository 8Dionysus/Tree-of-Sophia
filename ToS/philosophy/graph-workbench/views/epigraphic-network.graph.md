# Epigraphic Network Graph View

## Lens

Shows cities, inscriptions, carriers, alphabetic handoffs, public writing,
bilinguals, trilinguals, and distributed epigraphic networks.

## Future Inputs

- inscription, artifact, network, language, script, and medium nodes
- city, institution, and transmission nodes
- `inscribed-on`, `written-in`, `uses-script`, `transmits-to`, and
  `mediated-through` relations

## Boundary

This view tracks public-writing networks. It should not reduce alphabetic
handoff to a single origin story or a teleology toward later traditions.
