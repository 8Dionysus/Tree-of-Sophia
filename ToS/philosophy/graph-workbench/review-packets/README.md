# Review Packets

`review-packets/` defines what a graph review slice must carry for a human
operator or agent.

A review packet is not a verdict and not a canon promotion. It is a compact,
source-ref preserving view of one graph lens: counts, active filters, clusters,
unresolved diagnostics, weak source posture, hubs, isolated nodes, and the next
human review route.

`table-i-post-planting-audit.json` and `table-i-post-planting-audit.md` are
generated review packets for the current prepared Table I planting. They show
whether planted atlas rows, dossier branches, proposed graph rows, source
backlogs, and graph projection packets are ready for first human graph review.
