# Relation Kinds

`relation-kinds/` defines the empty vocabulary for future graph-readable ToS
relations.

These relation names keep ToS from collapsing every edge into generic
influence.

| Kind | Use |
| --- | --- |
| `belongs-to` | places an object inside a branch, corpus, tradition, institution, or context |
| `attested-in` | links a claim or object to a material or textual witness |
| `written-in` | links an object to a language or script-bearing textual form |
| `versioned-as` | links one text, decree, treaty, inscription, or corpus to a parallel language, recension, or presentation version |
| `inscribed-on` | links an inscription or text to stone, metal, clay, monument, sarcophagus, tablet, or other carrier |
| `sealed-by` | links archive, tablet, container, or administrative act to seal, seal impression, or sealing regime |
| `decreed-by` | links a decree or public text to the ruler, priesthood, city, institution, or authority that issued it |
| `legitimates` | links a text, ritual, inscription, or institution to the authority, succession, law, or order it authorizes |
| `ritualizes` | links a practice, legal act, oath, treaty, or concept to its ritualized performance layer |
| `uses-script` | links a witness, corpus, or tradition to a script |
| `uses-medium` | links a witness, corpus, or tradition to a material or publication medium |
| `preserved-by` | names survival through copy, archive, edition, quotation, excavation, or digitization |
| `copied-through` | names manuscript, scribal, scholastic, or print copying routes |
| `commentary-on` | links commentary, scholia, gloss, or interpretation to its object |
| `transmits-to` | points forward along a transmission corridor |
| `receives-from` | points backward along a transmission corridor |
| `develops-from` | names development without reducing it to direct influence |
| `parallel-to` | links structurally comparable objects without claiming descent |
| `contrasts-with` | marks a meaningful opposition or difference |
| `canonical-for` | links a text, corpus, figure, or relation to a community or canon posture |
| `provisional-for` | marks a scaffold as a candidate for later branch or canon review |
| `evidence-for` | links witness or scholarship to a claim |
| `contested-by` | links an object or claim to a controversy or contrary evidence |
| `deciphered-by` | links writing, corpus, or artifact to a decipherment route, scholar, method, or edition history |
| `partly-deciphered-as` | marks provisional readability without making a full semantic claim |
| `lost-but-attested-by` | links a lost corpus, language practice, or book culture to indirect witnesses, quotations, catalogues, or later testimony |
| `mediated-through` | marks access mediated by later editions, colonial recovery, excavation, translation, catalogue, or digital corpus |
| `translated-through` | names language, school, or institutional translation movement |
| `institutionalized-in` | links a practice, corpus, or concept to an institution or curriculum |

Relation kinds become canon only through reviewed relation packs.
