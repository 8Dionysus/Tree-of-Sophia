# Node Kinds

`node-kinds/` defines the empty vocabulary for future graph-readable ToS
objects.

These are object kinds, not populated nodes.

| Kind | Use |
| --- | --- |
| `civilization-node` | a civilizational or macrohistorical context when it is needed for branch placement |
| `tradition-node` | a tradition, school, disciplinary lineage, or branch of transmitted practice |
| `corpus-node` | a corpus, textual family, archive family, canon, or collection |
| `work-node` | an individual work or bounded textual object |
| `artifact-node` | a tablet, manuscript, inscription, codex, papyrus, seal, object, or material witness |
| `inscription-node` | an inscriptional object or inscription family whose public, monumental, archival, or dedicatory posture matters |
| `edition-node` | a critical edition, corpus edition, catalogue, or digital edition that mediates access without becoming the ancient source |
| `archive-node` | an archive, find complex, tablet group, dossier, or administrative corpus organized by preservation context |
| `decree-node` | a royal, priestly, civic, or institutional decree treated as a bounded object |
| `treaty-node` | a treaty, oath-bound agreement, diplomatic text, or versioned international instrument |
| `ritual-node` | a ritual prescription, liturgical action, cultic procedure, or ritualized textual practice |
| `decipherment-node` | a decipherment state, method, hypothesis, controversy, or research route around unreadable or partly readable writing |
| `version-node` | a language version, recension, translation version, or parallel inscriptional version |
| `network-node` | a city, trade, scribal, epigraphic, or transmission network when relations are distributed rather than single-line descent |
| `figure-node` | an author, editor, compiler, teacher, ruler, commentator, or named mask |
| `institution-node` | a school, temple, court, monastery, library, academy, university, press, journal, or archive |
| `language-node` | a language used by source, transmission, commentary, or reception |
| `script-node` | a writing system or script-bearing convention |
| `medium-node` | clay, papyrus, parchment, codex, print, journal, digital archive, or other carrier |
| `genre-node` | instruction, dialogue, law, commentary, omen series, treatise, aphorism, disputation, or other form |
| `concept-node` | a concept, term, semantic field, principle, or problem |
| `practice-node` | a knowledge practice, ritual practice, interpretive practice, discipline, or method |
| `chronology-node` | a period, formation window, fixation window, preservation event, or reception window |
| `transmission-node` | a translation route, commentary route, migration route, school route, or preservation corridor |
| `preservation-node` | a survival, copy, excavation, edition, digitization, or loss posture |
| `controversy-node` | a dispute, dating problem, attribution problem, decipherment problem, or interpretive risk |
| `source-witness-node` | a source-facing witness that belongs under source authority |
| `research-packet-node` | a non-authoritative scaffold that can feed review but cannot author truth |

Future branch work should pick the narrowest kind that keeps authority,
evidence, and graph projection legible.
