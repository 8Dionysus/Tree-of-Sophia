# Chronology

`chronology/` defines the time posture for future philosophy branch growth.

ToS chronology should keep several time layers visible:

| Layer | Use |
| --- | --- |
| formation | when a practice, text family, institution, or discourse begins forming |
| attestation | when a witness, artifact, manuscript, inscription, or publication is materially attested |
| fixation | when a text, corpus, recension, or curriculum stabilizes enough to cite |
| canonization | when later communities treat a corpus, work, figure, or school as authoritative |
| preservation | how the material survives, disappears, is copied, excavated, printed, digitized, or reconstructed |
| reception | when later branches receive, translate, contest, or reinterpret the material |

These layers are vocabulary for later review. They do not add dated corpus data
by themselves.
