# Evidence Status

`evidence-status/` defines how future branches speak about source certainty,
source limits, and graph readiness.

## Baseline Statuses

| Status | Use |
| --- | --- |
| `source-attested` | source witness or material witness can be named |
| `textually-stable` | a work, corpus, or recension has enough stability for branch review |
| `wisdom-or-normative-corpus` | the material carries reflective, ethical, legal, pedagogical, or normative thought without forcing a modern philosophy label |
| `technical-knowledge-corpus` | the material carries systematic knowledge practice such as commentary, medicine, divination, grammar, astronomy, or scholastic technique |
| `writing-attested-meaning-unconfirmed` | writing or signs are attested while semantic interpretation remains unresolved |
| `fragmentary-or-mediated` | survival, translation, quotation, excavation, or colonial mediation limits authority |
| `research-packet-only` | a lead exists only as non-authoritative research scaffolding |
| `canon-reviewed` | a node or relation has crossed the reviewed ToS canon route |

Evidence status constrains claims. It does not decide philosophical value by
itself.
