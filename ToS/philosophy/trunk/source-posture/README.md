# Source Posture

`source-posture/` defines the authority posture of material moving through ToS.

| Posture | Owner route |
| --- | --- |
| raw scaffold | `ToS/research-packets/` |
| branch interpretation | `ToS/philosophy/` |
| source witness | `ToS/source-witnesses/` or branch-local `sources/` before promotion |
| candidate node | `ToS/philosophy/graph-workbench/proposed-nodes/` or `ToS/candidate-intake/` |
| candidate relation | `ToS/philosophy/graph-workbench/proposed-relations/` or `ToS/candidate-intake/` |
| reviewed relation pack | `ToS/canon/relations/` |
| canon node | `ToS/canon/` |
| derived view | `ToS/derived-exports/` |

Generated graph views, compact indexes, and downstream exports remain lower
authority than the authored source route that produced them.
