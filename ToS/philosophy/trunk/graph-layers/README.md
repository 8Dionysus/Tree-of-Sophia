# Graph Layers

`graph-layers/` defines how ToS graph views can switch lenses without changing
source authority.

| Layer | Use |
| --- | --- |
| `source-relation` | witness, edition, translation, citation, or source-corpus anchoring |
| `historical-relation` | chronology, institution, medium, preservation, and reception movement |
| `conceptual-relation` | concepts, semantic fields, problems, contrasts, and analogies |
| `transmission-relation` | translation, copying, commentary, school, archive, and cross-branch corridors |
| `evidence-relation` | claims, disputes, uncertainty, preservation, and proof posture |
| `candidate-relation` | workbench relations before canon review |
| `canonical-relation` | reviewed relation packs under ToS canon |

Graph layers are projections over source-rooted tree growth. They do not
replace the branch, source witness, or canon route.
