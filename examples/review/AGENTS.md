# AGENTS.md

This file applies to review/archive material under `examples/review/`.

## What lives here

`examples/review/` keeps superseded scaffold and pilot examples that are no
longer part of the active compatibility surface.

It holds:

- review/archive material for earlier bounded pilots
- superseded example clusters kept for lineage of review rather than live canon
- archived example packs such as `examples/review/calibration-family-pilot/`

## Editing posture

Treat these files as review/archive material, not active canon.
Keep superseded surfaces visibly marked rather than silently deleted when they
still matter for review history.
Do not point current validators or tiny-entry routes at these files unless a
later pass explicitly reactivates them.
