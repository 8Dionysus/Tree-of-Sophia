# AGENTS.md

This file applies to primary witness and source files under `sources/`.

## What lives here

`sources/` holds raw source-facing witness material that feeds the authored ToS
tree.

These files may include:

- primary-language source witnesses
- bounded working or bridge translation witnesses
- source-linked donor markdown used to ground canonical tree nodes

## Editing posture

Keep files here source-facing, reviewable, and provenance-aware.
Keep `sources/` explicitly not node law.
Do not treat `sources/` as relation law or the public tiny-entry authority
surface.
Keep the difference visible between raw witness material in `sources/`, candidate
material in `intake/`, canonical authored nodes in `tree/`, and compatibility
surfaces in `examples/`.

When multilingual witness files are present:

- keep one shared source route rather than parallel language trees
- keep canonical-source versus translation posture explicit
- keep translator or editor attribution visible where it matters

## Validation

Use `docs/REVIEW_CHECKLIST.md` for broader review.
